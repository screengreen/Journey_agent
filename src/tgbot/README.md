# Telegram Bot

Бот для работы с каналами и созданием маршрутов.

## Установка

Убедитесь, что установлены все зависимости:
```bash
pip install -r ../requirements.txt
```

## Настройка

1. Создайте файл `.env` в корне проекта (используйте `.env.example` как шаблон)
2. Добавьте токен бота: `TELEGRAM_BOT_TOKEN=your_bot_token_here`

## Запуск

```bash
python bot/bot.py
```

## Функциональность

- **Добавление каналов**: Пользователь может добавить каналы Telegram по ссылке или названию
- **Создание маршрута**: Интерактивное создание маршрута с возможностью уточнений через комментарии
- **База данных**: SQLite база данных для хранения каналов пользователей

## Структура

- `bot.py` - основной файл бота (использует aiogram)
- `database.py` - работа с БД
- `agent_stub.py` - заглушка агентской системы (в будущем будет заменена на langgraph)

## Технологии

- **aiogram** - асинхронный фреймворк для работы с Telegram Bot API
- **FSM (Finite State Machine)** - для управления состояниями диалога
- **SQLite** - база данных для хранения каналов пользователей
